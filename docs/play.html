How to Play

SuperMario — AlgoNauts runs as a desktop app (Java required)

Download SuperMario.jar
1

Install Java 17+

You need Java 17 or newer installed on your machine.

Windows / Mac / Linux →
2

Download the JAR

Click the button above to download SuperMario.jar.

3

Run it

Open a terminal in the same folder and run:

java -jar SuperMario.jar

Or double-click the JAR if Java is set as the default handler.

4

Scores go online automatically

Your high scores are posted to the live leaderboard shared with all players worldwide.
