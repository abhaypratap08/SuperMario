# SuperMario — AlgoNauts

A retro-style platformer inspired by Super Mario, built in Java by Team AlgoNauts.

## Run

Requires Java 17+
```bash
java -jar SuperMario.jar
```

## Build from source
```bash
git clone https://github.com/abhaypratap08/SuperMario
cd SuperMario
./gradlew fatJar
java -jar build/libs/SuperMario.jar
```
